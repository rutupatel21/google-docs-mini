# google-docs-mini

This project requires:
* NodeJS version 6.0 or above
* MongoDB

Go to main directory and follow below steps:
* npm install
* node app.js

For demonstration: https://navigus-assessment-1.herokuapp.com/
REST API:
* / : Home page
* /register : sign up
* /login : login
* /fileUpload: to upload word document to the server
